About NodeSchool

Open source workshops that teach web software skills.

Our community host events in Chernivtsi for sharing of knowlendge and helping with going through the workshoppers.

Learn Coding

Join us and learn coding with us. You are going to learn the basics of Node.js. It's a plus if you know some JavaScript beforehand, but if not we can help you out.

Self-Guided

You will go through a set of self-paced challenges. No boring talks. (An occasional interesting talk is possible, though!) All challenges are Open Source and available online on nodeschool.io. This means you can continue learning after the event.

For Everyone

We think coding is awesome and everyone should feel welcome at our event. This includes that we are expecting all participants to be awesome to each other and respect the Code of Conduct.

Learn yourself

Workshoppers are self guided (you don't need to attend a workshop to do one) and most work offline.

Our workshoppers

There are a lot of useful workshoppers. Some of those are built in Chernivtsi, for our community.

learnyoubash

Learn you how to use the terminal and write your the first Bash script.

how-to-markdown

Learn how to start using Markdown — a lightweight markup language with plain text formatting syntax.

Mentors help you to find solutions

If you have a question, feel free to ask. We glad to help you.
